# webfejlesztesfinal
Radácsi Huba

https://radacsihuba.github.io/webfejlesztesfinal/

Trianoni kis ismeretterjesztő, történelmi kisokos. Területveszteség kalkulátor.

Két javascript file található. Egyik felelős a kalkulátor pontos működéséért, egyéni adat lekezelésére is, illetve a fájdalom mérő hozzákapcsolása a terület méretéhez. Második a kapcsolat oldal működéséért felelős, adatlap validálása, elküldés esetén visszajelzés, hibás adat esetén hibák jelzése, logolás a consoleba.
